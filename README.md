# Linea.rs
A basic rust linear algebra crate for personal use
## Project Structure

### Matrix
The matrix object provides the following methods:
- `from_vec`
- `fill`
- `get`
- `set`
- `row`
- `col`
- `is_square`
- `add`
- `multiply`
- `mul_vector`
- `transpose`
- `scalar_multiply`
- `scalar_add`
- `determinant`
- `inverse`
- `lu_decompose`
- `qr_decompose`
- `mul_vector`
